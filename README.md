## Задание на лабораторную работу

На основе [программы 3](https://github.com/eeeeagle/PM_3) реализовать один из алгоритмов балансиров-
ки деревьев.

**Примечание:** Балансировка дерева должна осуществляться на момент
вставки. Удаление элементов из сбалансированного дерева можно опу
стить. Также в программе должна быть предусмотрена возможность отоб
ражения текущей конфигурации дерева
